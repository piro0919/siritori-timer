# Siritori Timer

> Timer app for Japanese shiritori word-chain game.

限界しりとりタイマー。

[🔗 Live Site](https://siritori-timer.kk-web.link/)

## ✨ Features

- ⏱ Per-turn countdown for time-pressured shiritori
- 🔊 Audio cues (howler.js)
- 🎬 Smooth animations
- 📱 Installable PWA

## 🛠 Tech Stack

- Next.js + React + TypeScript
- Framer Motion
- Howler (audio)
- next-pwa

## 🚀 Development

```bash
pnpm install
pnpm dev
```
